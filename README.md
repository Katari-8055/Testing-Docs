# Multiplication Utility

This project provides a simple JavaScript utility function for performing multiplication between two numbers.  
The main purpose of this project is to demonstrate **clean code practices**, **basic module structure**, and **proper documentation using JSDoc**.

JSDoc helps developers understand how a function works by describing its parameters, return values, and usage examples. It also improves the developer experience by enabling **IntelliSense and type hints in editors like VS Code**.

---

# Features

- Simple and clean multiplication function
- Well documented using **JSDoc**
- Beginner-friendly JavaScript project
- Easy to integrate into other projects
- Demonstrates basic **Node.js module usage**
